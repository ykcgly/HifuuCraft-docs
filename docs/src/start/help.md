# 入门指南

## 云梦都服务器守则

:::info
如果您相信您收到了不正确的处罚或遇到了问题，请联系管理员
:::

在开始游玩之前，您需要了解到本服务器对玩家行为作出了一定规范。这包括：

- 本服务器允许 `Xaero's Minimap` 等辅助性模组，但是：
- 禁止外挂。这包括矿石透视（xray），杀戮光环等一系列第三方作弊功能
- 文明聊天，任何人不得以任何原因恶意骚扰或人身攻击其他玩家。如遇此情形，请呼叫管理员处理
- 禁止破坏其他玩家的游戏体验。这包括偷窃其他玩家物品，恶意破坏其他玩家建筑，锁其他玩家箱子等行为

违反者，依“善意推定原则”，予以处罚。  
形如“我不知道”的辩解并不总是一个有效的免责理由。服务器管理员对一切规则保留最终解释权。

## 正式游玩

### 1.1 准备开始

为了进入服务器，您需要游戏版本≥1.12.2的**Java版Minecraft**。您可以使用原版客户端，或是 Forge 和 Fabric 等模组加载器并安装 Sodium 等第三方模组来优化您的游戏体验。  

本服务器支持游戏版本向后兼容。这意味着您可以使用 Minecraft 1.21.8 进入服务器并游玩版本更低的 1.12.2 生存服，但是 Minecraft 1.14.4 客户端无法进入版本更高的 1.21 空岛服。

### 1.1.1 手机版玩家

如果您正在使用手机版 Minecraft 进入服务器，您可能需要QQ群聊内提供的整合包。

## 1.2 进入服务器

在启动游戏后，您便可以点击`多人游戏` -> `添加服务器` 并输入服务器地址。  
本服务器有多个IP地址：

:::warning
不要把IP地址后的空格与括号一并复制到服务器地址栏
:::

- `yunmengdu.cn` （主线路）
- `m.yunmengdu.cn:35565` （备用线路）
- `sblt.yunmengdu.cn` （联通专用）

输入服务器地址后点击`完成`并点击`加入服务器`即可。

## 1.3 服务器内

进入服务器后，您需要创建或登录账号。被创建/登录的账号取决于您的 Minecraft 游戏名。

输入密码后即可开启您的云梦都之旅。  
您可以使用物品栏中的`传送菜单（指南针）`或右键大厅内的NPC来进入对应的服务器。

祝您游玩愉快！
