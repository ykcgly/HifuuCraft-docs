# 常用指令

## 群内常用指令

### 1. /playerinfo

   玩家数据查询指令，群内使用。后面可以加上玩家名字（不用空格）或者直接@被查询人，后者会返回绑定的最后一个账号

### 2. /online

   查询服务器在线玩家

### 3. /msgs <文本>

   发送文本到服务器内
   ::: warning
   服务器非法外之地，请注意言辞
   :::

### 4. /checkbind

   查看绑定游戏账号情况，可以@被查询人使用，留空则查询自己的

### 5. /生成云梦都全身照

   直接输入可以生成自己的游戏全身照，依赖服务器皮肤生成，可以@被查询人使用

### 6. /setskin

  设置服务器皮肤。服务器有内置皮肤系统，离线玩家可以使用该指令设置离线皮肤，使用后上传64*64的皮肤文件，必须为.png格式(记得使用原图上传)

## 服务器内常用指令

### 1. /bindqq <你的qq号>

绑定你的QQ账号，可以用于账号找回以及数据查询
==(一定要绑定啊)==

### 2. /bindqq clear

  解绑指令，解绑最后一个已经绑定的玩家账号，一般情况用于换绑，也可以跟上要解绑的号的名字
  ::: warning
   如非必要请勿解绑！
  :::

### 3. /skin set

你的正版名字 要是你之前的离线账号登录，换成正版账号登录后可能出现皮肤不刷新问题，可以使用后该指令刷新皮肤
（在子服中使用可能导致断开连接并且回到大厅）

::: tip
当你尝试上述指令无效时，可以试试这个
/skin clear
:::
